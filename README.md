# RicSan's Apartment Showcase

Full-stack luxury apartment showcase - Laravel 11 + React 18

### Features:
- 6 luxury apartment listings
- Fully responsive (mobile to desktop)

### Tech Stack:
- Laravel 11 + MySQL
- React 18 + Vite
- Tailwind CSS v4
- Context API
- Axios

### Development:
```bash
php artisan serve
npm run dev
```
